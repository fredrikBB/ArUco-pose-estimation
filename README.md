# Prerequisites
* Assummes the host has installed OpenCV (See https://docs.opencv.org/4.x/d7/d9f/tutorial_linux_install.html).
* CMake
* gcc

# How to build and run the project
1. cd <project_folder>
2. cmake .
3. make
4. output/<name_of_executable> <list_of_args>
